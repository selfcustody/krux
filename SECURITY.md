# Security policy

Please report any vulnerability or any bug that could potentially affect the security of users
funds by mail to
[`odudex@proton.me`](mailto:odudex@proton.me)
or
[`jean.dominique.lecomte.delisle@protonmail.me`](mailto:jean.dominique.lecomte.delisle@protonmail.me).

You may use `odudex` or `jean` PGP public key to encrypt your mail with the
`<key>` (odudex)
or
`<key>` (jdlcdl).

You can get both from:

**odudex**

- [`keys.openpg.org`](https://keys.openpgp.org/search?q=m%40odudex): `gpg --keyserver hkps://keys.openpgp.org --receive <key>`.
- [`keyserver.ubuntu.com`](https://keyserver.ubuntu.com/search?q=m%40odudex): `gpg --keyserver hkps://keyserver.ubuntu.com` --receive <key>`.

**jdlcdl**

- [`keys.openpg.org`](https://keys.openpgp.org/search?q=m%40jdlcdl): `gpg --keyserver hkps://keys.openpgp.org --receive <key>`.
- [`keyserver.ubuntu.com`](https://keyserver.ubuntu.com/search?q=m%40jdlcdl): `gpg --keyserver hkps://keyserver.ubuntu.com` --receive <key>`.

In the subject type `[Krux firmware] Security Report: <short description>`
and in the body a long description describing the issue.
