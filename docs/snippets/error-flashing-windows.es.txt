En Windows, al intentar flashear Yahboom o WonderMV usando **Krux-Installer**, puede aparecer el error **flash failed**. esto significa que Windows no encuentra el controlador para comunicarse con el dispositivo por USB (aparecerá un dispositivo *USB Serial* no reconocido en *Administrador de dispositivo -> Otros dispositivos*). Para solucionarlo, descargue e instale el controlador [CH340/341](https://www.wch-ic.com/downloads/ch341ser_exe.html) (aparecerá un dispositivo *USB-SERIAL CH340* en *Administrador de dispositivos -> Puertos (COM & LPT)*).
